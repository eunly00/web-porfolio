Crop Quality Assessment Using Image Processing and Artificial Intelligence

AI & COMPUTER VISION 2019.09-2020.08

This project developed an AI-based crop quality assessment system designed to detect plant diseases and evaluate leaf health efficiently in smart farming environments. The system successfully transitioned from unsupervised visual grounding models (GroundingDINO + SAM) to a high-performance YOLOv8-based supervised framework.
