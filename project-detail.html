Crop Quality Assessment Using Image Processing and Artificial Intelligence

AI & COMPUTER VISION 2019.09-2020.08

This project developed an AI-based crop quality assessment system designed to detect plant diseases and evaluate leaf health efficiently in smart farming environments. The system successfully transitioned from unsupervised visual grounding models (GroundingDINO + SAM) to a high-performance YOLOv8-based supervised framework.

기술 스택

Python YOLOv8 GroundingDINO SAM Computer Vision

주요 기능

  • 식물 질병 자동 감지 시스템
  • YOLOv8 기반 고성능 객체 탐지
  • 실시간 작물 품질 평가
  • 스마트 팜 환경 최적화

프로젝트 개요

2024학년도 2학기 5W 캡스톤디자인 경진대회

전북대학교 산학실전캡스톤 프로젝트로, 스마트팜 환경에서 작물의 잎 개수를 정확하게 파악하고, 비정상 잎(질병 및 병충해)을 탐지하기 위한 경량화된 객체 탐지 모델을 개발했습니다.

개발 동기 및 목적

개발 동기

도시농업의 급격한 성장과 함께 농업의 접근성과 효율을 높이기 위한 스마트 기술에 대한 필요성이 증가했습니다. IoT 기술과 인공지능을 결합한 스마트 팜 시스템을 구축하고 식물의 생육상태를 효율적으로 판별할 수 있는 AI 기반 품질 판별 시스템 개발이 필요했습니다.

프로젝트 목적

  • AI 기술과 이미지 프로세싱을 활용해 식물의 생장 상태와 품질을 정량적으로 판별하여 작물 관리의 효율성을 높이기
  • 식물의 생장 정도를 파악하기 위해 잎의 개수 측정
  • 식물의 질병 유무를 파악하기 위해 잎의 상태를 보고 질병 유무 파악

개발 과정

초기 개발 (1단계)

  • GroundingDINO와 SAM 기술을 사용해 텍스트 프롬프트를 기반으로 잎 영역을 탐지
  • 탐지된 잎의 평균 RGB값을 산출하여 잎의 정상/비정상 여부 판별
  • 문제점: RGB 분석 방식은 비정상 잎이 정상 잎보다 많은 데이터 불균형에 민감해, 평균 값이 왜곡되어 정상 잎이 비정상 잎으로, 비정상 잎이 정상 잎으로 판별되는 문제 발생

기술 전환 (2단계)

  • 기존 데이터 외에 웹 크롤링, Kaggle에 있는 open source 데이터를 추가로 수집
  • RoboFlow를 사용해 데이터셋 라벨링 및 증강 (약 60배 이상의 데이터 증량)
  • 고속 객체 탐지 모델인 YOLOv8을 이용해 학습 진행
  • 학습 환경을 Colab(GPU)과 개인 CPU로 나누어 학습 진행
  • 학습된 모델로 잎의 상태를 분석하고 탐지된 박스의 개수를 통해 잎의 개수 판별

주요 성과

  • 정확도: 평균 mAP@0.5: 0.810
  • Diseased 클래스 AP: 0.846
  • Healthy 클래스 AP: 0.774
  • 왼쪽 상단에 정상 잎과 비정상 잎의 개수를 시각적으로 표시
  • 오른쪽 상단에 정상 잎과 비정상 잎의 비율을 계산하여 건강상태를 시각적으로 표시 (Normal, Caution, Warning, Danger, Critical)
  • 인트초코뿐만 아니라 다른 품종에도 적용 가능하도록 설계
